Criterios de evaluación

Para llevar a cabo la generación de informes basados en los registros y tablas almacenados en MySQL, empleamos el complemento de generación y exportación de datos de MySQL a Excel. Este complemento se instala de manera sencilla al ser parte integral de la estructura de directorios de la aplicación. En este contexto, se definen secciones que permiten importar datos y, simultáneamente, construir y gestionar la interacción con la base de datos MySQL.

Una vez que hayamos exportado los datos desde MySQL a Excel, estaremos en condiciones de generar tablas dinámicas y gráficos. Estas herramientas nos proporcionarán una comprensión más profunda y permitirán un análisis didáctico de cómo se relacionan e interactúan los datos presentes en las tablas.

Encriptación

Spring Boot Security ofrece varios algoritmos de encriptación, siendo BCrypt el que se uso en el proyecto DNQ Motor. Cuando un usuario crea o cambia su contraseña, esta se almacena cifrada en la base de datos. Durante el inicio de sesión, Spring Boot Security compara la contraseña proporcionada con la versión cifrada en la base de datos, lo que garantiza la seguridad de las contraseñas. Además, se utiliza el "salting" para agregar una capa adicional de seguridad, y las contraseñas se actualizan de manera segura.
